Course Projects

Computer Vision (ECE 5554)

  • Estimating GPS coordinates from the an image: Estimating geographic location information from an image has received a lot of attention from the computer vision community. This is a task which is not only difficult for machines but for humans as well. We have developed a system that can be used by crowdworkers to collectively arrive at consensus regarding the location of a query image. In the process the crowdworker is assisted by content-based image retrieval techniques. [Project webpage]

Machine Learning (CS 5824)

  • Birds in a Forest: We have used a random forest approach for bird species identification. First we train 25 SVM's to predict 25 bird features for our data set images. Then we train a random forest to identify the specific bird species. We used the CUB-200-2011 bird data set for this task. [Project report]
